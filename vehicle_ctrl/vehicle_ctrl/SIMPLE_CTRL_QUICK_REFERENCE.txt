📋 核心函数（必须理解）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. control_loop_improved()  [549 行] ⭐⭐⭐⭐⭐
   作用: 主控制循环，计算并发送车辆控制命令
   频率: 15 Hz
   流程: 检查状态 → 找目标点 → 计算速度 → PID 控制 → 平滑 → 发布

2. find_target_waypoint_improved()  [416 行] ⭐⭐⭐⭐⭐
   作用: 智能选择前瞻目标点
   算法: 自适应前瞻距离 + 弯道检测
   返回: 目标路径点索引

3. waypoints_callback()  [179 行] ⭐⭐⭐⭐⭐
   作用: 接收路径并转换为 CARLA 格式
   处理: ROS Path → CARLA Transform 列表
   特点: 找最近点作为起始索引

4. odometry_callback()  [156 行] ⭐⭐⭐⭐⭐
   作用: 接收车辆位置、速度、姿态
   频率: 50-100 Hz
   数据: 位置(x,y,z) + 姿态(roll,pitch,yaw)

5. run_optimized()  [724 行] ⭐⭐⭐⭐⭐
   作用: 状态机主循环
   状态: 等起点 → 等目标 → 等路径 → 控制 → 停止

🎮 控制相关函数
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

6. setup_PID_improved()  [335 行] ⭐⭐⭐⭐
   作用: 创建并配置 PID 控制器
   参数: K_P, K_I, K_D (横向和纵向)
   返回: VehiclePIDController 实例

7. smooth_steering_improved()  [479 行] ⭐⭐⭐⭐
   作用: 平滑转向角，避免抖动
   方法: 指数平滑 + 曲率自适应
   效果: 弯道快速响应，直道平滑行驶

8. calculate_turn_speed()  [494 行] ⭐⭐⭐⭐
   作用: 根据弯道程度计算安全速度
   逻辑: 曲率越大 → 速度越低
   范围: 20-50 km/h

9. apply_gradual_braking()  [535 行] ⭐⭐⭐
   作用: 接近终点时渐进刹车
   策略: <5m 强刹, <10m 中刹, <20m 轻刹

10. start_control()  [694 行] ⭐⭐⭐⭐
    作用: 初始化并启动控制定时器
    工作: 重置状态 + 创建 PID + 启动 15Hz 定时器

11. emergency_control()  [511 行] ⭐⭐⭐
    作用: 紧急停车
    命令: 油门=0, 转向=0, 刹车=0.5

12. calculate_path_curvature()  [371 行] ⭐⭐⭐⭐
    作用: 计算前方路径的弯曲程度
    方法: 采样 5 个点，计算平均角度变化
    返回: 曲率（度）

13. calculate_adaptive_lookahead()  [397 行] ⭐⭐⭐⭐
    作用: 根据曲率动态调整前瞻距离
    逻辑: 急弯 7m, 缓弯 12m, 直道 14m
    范围: [3m, 8m]

14. check_destination_reached()  [518 行] ⭐⭐⭐
    作用: 判断是否到达目的地
    条件: 最后一个路径点 AND 距离 < 3m
    返回: True/False

15. initialpose_callback()  [254 行] ⭐⭐⭐⭐
    作用: 接收起点，放置车辆但不启动
    逻辑: 存储起点 + 设置等待模式 + 停止旧控制
    触发: RViz "2D Pose Estimate"

16. goal_pose_callback()  [265 行] ⭐⭐⭐⭐⭐
    作用: 接收终点，允许车辆启动
    逻辑: 存储终点 + 取消等待 + 重置状态
    触发: RViz "2D Nav Goal"

17. calculate_distance()  [365 行] ⭐⭐⭐
    作用: 计算两点间距离
    公式: sqrt((x1-x2)² + (y1-y2)²)
    调用: 非常频繁（每周期多次）

18. create_ctrl_msg()  [239 行] ⭐⭐⭐
    作用: 创建 ROS 控制消息
    处理: 限制范围 + 格式转换
    输出: CarlaEgoVehicleControl

19. get_spectator_transform()  [283 行] ⭐⭐
    作用: 计算跟随相机位置
    位置: 车后 8m，高 4m
    用途: 可视化

20. update_spectator_view()  [310 行] ⭐⭐
    作用: 更新 CARLA 观察者视角
    方法: 后台线程，非阻塞
    频率: 20 Hz（限流）

启动阶段:
  main() → __init__() → run_optimized()

等待起点:
  run_optimized() 循环等待
      ↓
  initialpose_callback() → 车辆放置，等待模式

等待终点:
  run_optimized() 显示 "等待目标点..."
      ↓
  goal_pose_callback() → 允许启动

等待路径:
  run_optimized() 显示 "等待路径规划..."
      ↓
  waypoints_callback() → 路径数据就绪

控制阶段:
  run_optimized() → start_control() → 启动定时器
      ↓
  control_loop_improved() [15 Hz]
      ├─ find_target_waypoint_improved()
      ├─ calculate_turn_speed()
      ├─ PID.run_step()
      ├─ smooth_steering_improved()
      └─ 发布控制命令

到达目标:
  check_destination_reached() → 渐进刹车 → 停止
